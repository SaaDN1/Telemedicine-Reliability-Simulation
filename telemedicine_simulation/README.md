
Telemedicine Simulation - React (minimal)
Files:
- public/index.html
- src/index.jsx
- src/App.jsx
- package.json

Run locally:
1. Install Node.js and npm.
2. From project root: npm install -g parcel-bundler
3. npm install
4. npm start

This simple demo simulates appointment booking and unreliable video connections with a fallback to chat, and includes a Humanization checklist component.
